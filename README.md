# QA Feedback Formatter

A web-based tool designed to streamline QA feedback submission by automatically converting shorthand pricing issue notes into properly formatted text ready for JotForm.

## Features
- Convert shorthand ASIN pricing feedback into standard format
- Support for multiple lots in a single conversion
- One-click copy to clipboard
- Clean, user-friendly interface
- No installation required - runs entirely in the browser

## Use Case
Quality assurance teams can quickly document pricing discrepancies by entering shorthand notes (lot name, ASIN, manifest price, correct price) and instantly receive formatted feedback text that matches organizational standards.

## Usage
Simply paste your shorthand feedback, click format, and copy the result to your JotForm.
